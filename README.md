# stage2-template
Same layout as stage1-template: grading workflow, directory structure for problems and solutions.

**Stage 2** (Problems 1–10): recursive problems. Each has `starter.cpp`, `starter.py`, and `starter.java` in `problems/problemN/`, with tests in `tests/problemN/`.

- Problem 1: GCD (Euclid's algorithm, recursive)
- Problem 2: Factorial (recursive)
- Problem 3: Sum of digits (recursive)
- Problem 4: Power x^n (recursive)
- Problem 5: Fibonacci (recursive)
- Problem 6: Binary search (recursive)
- Problem 7: Decimal to binary (recursive)
- Problem 8: Max and min in array (recursive)
- Problem 9: All subsets (recursive)
- Problem 10: Palindrome check (recursive)
