# Problem 5 — Fibonacci

Implement the Fibonacci sequence recursively. Note F(0)=0, F(1)=1 is a constraint

## Input Format

- One non-negative integer n (index)

## Output Format

A single integer: F(n) (the nth Fibonacci number)

## Example

Input:
7

Output:
13

## Constraints

- n is a non-negative integer
- F(0)=0, F(1)=1

## Expected Complexity

- Time: O(2^n) naive recursion; O(n) with memoization
- Space: O(n) recursion stack
