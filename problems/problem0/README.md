# Problem 0 — Palindrome Check

Determine if a string is a palindrome using recursion. 

## Input Format

- One string (no spaces, single word)

## Output Format

True or False

## Example

Input:
racecar

Output:
True

## Constraints

- String contains only alphanumeric characters

## Expected Complexity

- Time: O(n)
- Space: O(n) recursion stack
