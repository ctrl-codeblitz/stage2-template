# Problem 1 — GCD (Greatest Common Divisor)

Implement the GCD of two non-negative integers using Euclid's algorithm recursively. 

## Input Format

- Two integers a and b, separated by whitespace
- They may appear on the same line or on separate lines

## Output Format

A single integer: the GCD of the two integers

## Example

Input:
48 18

Output:
6

## Constraints

- Integers are non-negative integers

## Expected Complexity

- Time: O(log min(a, b))
- Space: O(log min(a, b)) recursion stack
